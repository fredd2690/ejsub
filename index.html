¿Cómo usar ésta aplicación?

Ésta app le permitirá calcular el número de hosts posibles para las subredes, ó el número de subredes para contener determinado número de hosts Ésta App necesita 2 datos básicos y uno opcional:

a) Una dirección IP válida en formato decimal.

b) El número de subredes (si desea saber el número de hosts) ó el número de hosts (si desea obtener el número de subredes)

c) Una máscara de red en formato decimal (opcional), si no lo escribe, ésta app asignará una máscara por defecto, basándose en la clase a la que petenece la IP dada.
